---------------------------------------------
✅ 0단계: 프로젝트 압축 해제
---------------------------------------------

1. 받은 'project.zip' 파일을 우클릭하고 [모두 압축 풀기]를 클릭합니다.
2. 압축을 푼 후 생성된 폴더 경로를 기억해둡니다. (예: C:\\Users\\minju\\Downloads\\project)


---------------------------------------------
✅ 1단계: 명령어 입력창(cmd 또는 아나콘다) 열기
---------------------------------------------

아래 중 하나를 실행합니다:

- 윈도우 키 → "cmd" 검색 → [명령 프롬프트] 실행
- 또는 Anaconda Prompt, PowerShell Prompt 등 사용


---------------------------------------------
✅ 2단계: 아래 명령어를 1줄씩 입력
---------------------------------------------

📌 아래의 경로는 본인의 실제 project 폴더 위치에 맞게 수정해주세요.

① project 폴더로 이동

cd C:\\Users\\minju\\Downloads\\project

② 가상환경 만들기

python -m venv venv

③ 가상환경 활성화

venv\\Scripts\\activate

→ 활성화되면 커맨드 줄 앞에 (venv)가 붙습니다.
예: (venv) C:\\Users\\minju\\Downloads\\project>

④ 필요한 패키지 설치

pip install -r requirements.txt

→ 설치는 1~2분 정도 소요됩니다.


---------------------------------------------
✅ 3단계: 웹앱 실행
---------------------------------------------

아래 명령어 입력:

streamlit run app.py

→ 웹 브라우저가 자동으로 열립니다.

주소:
http://localhost:8501


---------------------------------------------
✅ 오류가 나는 경우 확인
---------------------------------------------

- Python 설치 안 됨 → https://www.python.org/downloads/
- pip 명령어가 안 됨 → 환경변수 설정 확인
- secrets.toml 오류 → .streamlit 폴더 안에 API 키가 입력되었는지 확인
